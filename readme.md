# 这是什么？
- 这是一个基于JupyterLab的控制面板
# 可以干什么？
- ①配合HuggingFace或InterDevCloud开服务器
- ②当做一个测试的服务器用（没有root权限）
- ③其他任何不需要root的事？？？
# 要注意的地方？
- 默认密码：password
- 修改密码请修改“start_server.sh”中的“JUPYTER_TOKEN”
- HuggingFace仅开通7860端口
- InterDevCloud可以自选一个端口，请修改“start_server.sh”中的“port”项
